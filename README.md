# faker.app
Generate fake data using Faker.js in a Node application. generated from [here](https://github.com/justusmbuvi/Faker.js-node-application)

## Steps

- Initialize project
- Configure nodemon in package.json
- Add a starting point
- Add models folder
- Add a views folder
- Start MongoDB
- Run the project
